Custom Instructions Interface

Last updated Apr 26, 2011

The NIOS 2 communicates to the Array Processor Unit (APU) through a custom instruction interface. The instruction format has an "n" port.

  • n=0 instructions are array processor instructions that are passed into the array processor instruction FIFO.
  • n=1 instructions are not meant for the array processor. They are about the status of the array processor.

n=1 Instructions

dataa Description
0 Busy

Returns 1 to NIOS 2 if the array processor is busy.
1 Elements in instruction FIFO

Returns the number of elements in the "instr" FIFO.
2 Elements in result FIFO

Returns the number of elements in the "result" FIFO.
3 Result

Read an element from the "result" FIFO.
