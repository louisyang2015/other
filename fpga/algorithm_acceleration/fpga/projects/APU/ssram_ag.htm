Configuration

Register loading occurs with "load" = 1.

load_mode[1:0] Action Performed
0 rdaddress <= address_in;
1 skip <= address_in;
2 highest_addr <= address_in;
3 block_size <= address_in;

 

Counting

assign next_addr = {1'b0, rdaddress} + {skip[N-1], skip};

if(next_addr > {1'b0, highest_addr})
    rdaddress <= rdaddress - block_size;
else
    rdaddress <= next_addr[N-1:0];
