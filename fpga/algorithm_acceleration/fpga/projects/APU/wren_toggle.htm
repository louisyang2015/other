Wren_Toggle

Last updated Apr 25, 2011

 

Mode[1:0] Description
0 OFF
wren_r and wren_i will both be zero even if "run" is 1
1 Real Only
wren_r will be 1 when "run" is 1
2 Imaginary Only
wren_i will be 1 when "run" is 1
3 Both Real and Imaginary
wren_i and wren_r will both be 1 when "run" is 1

So effectively:

  • mode[0] = real on
  • mode[1] = imag on

During the instruction decode phase, the Wren_Toggle is set up so that only the correct write enable signals (wren) will go high later on.
