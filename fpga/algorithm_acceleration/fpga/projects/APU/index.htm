Quartus / ModelSim Difference

  • AP_Control.v's parameterization (param) is done differently to avoid a warning in Quartus
  • APU_ALU.v - the "pi_int_k" block is connected different due to a last minute discovery of the AEC algorithm's need. The Quartus version has a 14 bit left shift. Both ModelSim and Quartus versions are in the same file, but commented out as needed.
