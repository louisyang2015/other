Instruction from NIOS 2

Custom instructions to allow NIOS 2 to interact with the FFT block.

dataa[10:8] dataa[7:0] datab[31:0] Description
1 address data

Write "data" into memSr[address].
2 address data Write "data" into memSi[address].
3 address X Read memSr[address] and return its content.
4 address X Read memSi[address] and return its content.
5 0 X Do FFT on the data current present at memSr and memSi.
5 1 X Do inverse FFT on the data current present at memSr and memSi.
The divide by N portion of inverse FFT has not been implemented.
6 X X Return 0 if FFT module is idle - meaning done with FFT. Return non-zero if busy.
