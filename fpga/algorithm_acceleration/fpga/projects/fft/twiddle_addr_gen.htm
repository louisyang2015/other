Twiddle Factor Address Generator

module Taddress_GEN(input clk, start,
        output reg [6:0] t_address);

 

The only mode is to provide address for the FFT twiddle factor read.

  • For the first 128 clock cycles, t_address is 0.
  • For the second 128 clock cycles, t_address is 0, 64, 0, 64, 0, 64, ...
  • For the third 128 clock cycles, t_address is 0, 32, 64, 96, 0, 32, 64, 96, ...
  • and so on...
  • For the final 128 clock cycles, t_address is 0, 1, 2, 3, ... 127
