I/O Pin Assignments

Go to Assignments menu --> Pin Planner. Export the pins listed as "CSV" file. Open the file with Excel. Remove the comments at the top of the CSV file. Use Excel to copy in the pin assignments from other sources. After the pin assignments are filled in, import the new CSV file into Quartus 2.

If you need to reverse the content of a column in Excel, check out: reverse_example.xlsx

At the end of I/O assignment, inside the pin planner, run Processing menu --> Start I/O Assignment Analysis

Global I/O Pin Settings

Pin capacitance: Assignments menu --> Device. Press the "Device and Pin Options" button. In the category pane, choose "Capacitive Loading". Assign 10pF to 3.3V LVTTL.

In Pin Planner, this can be overridden on a per pin basis by right clicking in the window with the table of pins, and choosing "customize columns".

Unused Pins: Same "Device and Pin Options" button as above. In the category pane, choose "Unused Pins". Select "As input tri-stated".

Dual-Purpose Pins: If necessary, choose "used as regular I/O".

Individual I/O Pin Settings

This is done via Assignments menu --> Assignment Editor.

The following are needed for output pins:

  • Slew Rate: 0 = slow, 2 = fast (default)
  • Current Strength, for 2.5V I/O: 4, 8, 12, or 16 (mA)
