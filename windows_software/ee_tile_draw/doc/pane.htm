Pane

The Pane class is a lot like the Grid class --- just fewer options and much easier to program.

Size:

The pane is always 2 rows by 5 columns

Data Structure:

The cells are indexed left to right, up to down

An array of integers (size 10 of course) stores the component ID to be shown at each cell.

A number stores the current selection (0 through 9)

A Rect "drawing_area" stores the rectangle bounding the entire grid.

Integer arrays "x_coords" and "y_coords" stores the pixel coordinates that is necessary to draw the grid lines (These are 3 x numbers and 6 y numbers referencing the red points of this grid)

At construction time, the Pane object will initialize its internal color variables and will use those same colors for the duration of the program.
