Inserting Components

Left click to select, Right click to add

Left click on the blue pane on the left to select a component

Then right click on the grid on the right to insert the component

Use the "Del" key to clear up tiles

 

Use the insert menu to insert components

The first time you select an different item the blue pane changes

 

If you select the same item again that item will be inserted

 

Use arrow keys to move beyond the current screen

  • move off the edge of the screen with the arrow key
  • jump an entire screen with ctrl-arrow key

 

There are often two or more ways to accomplish the same task.

Most of the menus items have short cut keys:

 

Example: There are three ways to rotate a tile

you can click on the blue arrow on the pane

or you can select the menu item

or you can press the F4 key
