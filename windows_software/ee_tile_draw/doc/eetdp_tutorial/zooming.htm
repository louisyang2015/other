Zooming and Saving your Picture

Use the "Zoom Level" menu to adjust how many tiles appear on screen

Use the "File menu" to save the drawing

 

The size of the bitmap or GIF generated is based on how big the components appear on screen. So if you want the resistors and transistors to appear small in the bitmap generated, zoom all the way out before using the "Save as" option.
