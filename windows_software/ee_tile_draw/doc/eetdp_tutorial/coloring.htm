Coloring

No drawing program is complete without some kind of coloring feature

You choose colors via the "Tile Options" menu

After you selected a new foreground and/or background color, the next tile that you draw will use these colors

You can also choose the "Apply Color to Current Tile" option (keyboard: F3) to change the color of any tile on screen --- including tiles that are blank

 

There are components that will occupy more than one tile, and in these cases the background color is applied only to the tile of insertion.

You should use the F3 key to manually apply the color to other tiles as needed.
