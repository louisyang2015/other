Transformations

You do transformations by calling "Graphics" class's methods.

scaling transformation:

graphics.ScaleTransform(1.75f, 0.5f);
//expands the rectangle by a factor of 1.75 in the x direction
//and shrinks the rectangle by a factor of 0.5 in the y direction

Rotation:

graphics.RotateTransform(28.0f); //rotate by 28 degrees

Translation:

graphics.TranslateTransform(150.0f, 150.0f);

Internally, calling each of the above methods will update a tranformation matrix inside the "Graphics" object.
