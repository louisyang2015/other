Using a Brush to Fill Shapes

Brush:

SolidBrush brush(Color(255, 255, 0, 0)); //Color(alpha, red, green, blue)

 

Ellipse:

graphics.FillEllipse(&brush, 0, 0, 100, 60);

 

Other Notes:

Hatch Brush:

The following creates a light blue brush with red horizontal lines:

HatchBrush brush(HatchStyleHorizontal, Color(255, 255, 0, 0),
        Color(255, 128, 255, 255));
//HatchBrush(style, hatch color, brush color)
//for more choices, see the HatchStyle enumeration

 

Texture Brush:

The following creates a brush that's based on an image

Image image(L"ImageFile.jpg");
TextureBrush tBrush(&image);
tBrush.SetTransform(&Matrix(75.0/640.0, 0.0f, 0.0f,
   75.0/480.0, 0.0f, 0.0f));

assuming the original picture is 640x480, the "SetTransform" scales it to 75x75

graphics.FillEllipse(&tBrush,Rect(0, 150, 150, 250));

When you draw with a texture brush and the drawing area is larger than the brush area, there will be tiling of the brush image. You can flip the image during this tiling:

tBrush.SetWrapMode(WrapModeTileFlipX); //flip horizontally
tBrush.SetWrapMode(WrapModeTileFlipY); //flip vertically
tBrush.SetWrapMode(WrapModeTileFlipXY); //flip both horiz, vert
