Constructing and Drawing Paths

A path is a sequence of graphics primitives (lines, rectangles, curves, text, and the like) that can be manipulated and drawn as a single unit.

Pen pen(Color(255, 255, 0, 0));
GraphicsPath path;
path.AddArc(175, 50, 50, 50, 0, -180);
//others: AddLine, AddCurve, AddRectangle, AddEllipse

graphics.DrawPath(&pen, &path);

A path is be divided into figures.

Say you do:

path.AddLine(...
path.AddLine(...
path.AddRectangle(...
path.AddEllipse(...

There are 3 figures generated --- The first figure consists of the two lines, the second figure consists of the rectangle, and the third figure consists of the ellipse.

Graphics::FillPath

The FillPath method fills the path according to the fill mode (alternate or winding) currently set for the path. The default mode is FillModeAlternate

graphics.FillPath(&brush, &path);
