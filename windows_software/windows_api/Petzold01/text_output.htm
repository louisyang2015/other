Text Output

TextOut:

TextOut(hdc, x, y, psText, iLength);

To specify alignment:

SetTextAlign(hdc, TA_RIGHT | TA_TOP);

Windows will interpret the coordinates passed to subsequent TextOut functions as specifying the top-right corner of the txt string rather than the top left corner

When printing text, to preserve the current background:

SetBkMode (hdc, TRANSPARENT);

An alternative to TextOut is to use "DrawText"

 

Information about the current font:

TEXTMETRIC tm;
hdc = GetDC(hwnd);
GetTextMetrics(hdc, &tm);
ReleaseDC(hwnd, hdc);

The fields of the TEXTMETRIC:

typedef struct tagTEXTMETRIC
{
    LONG tmHeight;
    LONG tmAveCharWidth;
    LONG tmMaxCharWidth;
}TEXTMETRIC, *PTEXTMETRIC;

these fields are in logical coordinates.

To get information about the system:

GetSystemMetrics(SM_CXSCREEN);

To see other possibilities, run the author's program "SysMets3"

 

Fonts:

To use the basic fixed-pitch font of the system:

SelectObject (hdc, GetStockObject (SYSTEM_FIXED_FONT));

Other options: SYSTEM_FIXED_FONT, OEM_FIXED_FONT

The font files have *.FON extensions.

The CreateFont() function has a lot of arguments.
