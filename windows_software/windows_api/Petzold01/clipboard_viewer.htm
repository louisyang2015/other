Other notes:

A program that is notified of changes in the clipboard contents is called a "clipboard viewer".

Windows can have a whole chain of clipboard viewers. But when the clipboard change, Windows notify just one clipboard viewer, and that clipboard viewer would notify the next one in the chain, and so forth.

When a program registers itself as a clipboard viewer, Windows gives that program the window handle of the previous "head of the clipboard viewer chain".

To register as a clipboard viewer:
hwndNextViewer = SetClipboardViewer(hwnd);

After registration, the program will get the WM_DRAWCLIPBOARD message whenever the content of the clipboard changes.

To leave the clipboard chain:
ChangeClipboardChain (hwnd, hwndNextViewer);

If someone is dropping out of the chain, Windows will send WM_CHANGECBCHAIN message to the start of the clipboard chain.

A clipboard viewer that handles data formats beyond the standard formats has a lot more work to do.
