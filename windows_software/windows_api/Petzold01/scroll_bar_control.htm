The Scroll Bar Control

click here for a program that demonstrates using a horizontal scrollbar control

you create this with the predefined window class "scrollbar", and one of these two styles: SBS_VERT and SBS_HORZ.

Scroll bar controls don't send WM_COMMAND messages, they send WM_VSCROLL and WM_HSCROLL messages.

Setting up a scroll bar control is also the same as setting up a window scroll bar --- except the second argument is now different:
SetScrollInfo (hwndScroll, SB_CTL, &si, bRedraw);
