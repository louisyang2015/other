Getting the Current Date and Time

SYSTEMTIME st;
GetLocalTime (&st);

 

The SYSTEMTIME Structure has these fields:

WORD wYear ;
WORD wMonth ;
WORD wDayOfWeek ;
WORD wDay ;
WORD wHour ;
WORD wMinute ;
WORD wSecond ;
WORD wMilliseconds ;

 

Other Notes:

If you want the international support from Windows, you can format a date or time using the "GetDateFormat" and "GetTimeFormat".
