The Button Controls

About Button Creation:

buttons have the Windows class name: "button"

Push buttons have the windows style: BS_PUSHBUTTON or BS_DEFPUSHBUTTON

Check boxes have the window style: BS_CHECKBOX or BS_AUTOCHECKBOX

Radio buttons have the window style: BS_RADIOBUTTON or BS_AUTORADIOBUTTON

Tri-State check boxes have the window style: BS_3STATE or BS_AUTO3STATE

Group boxes have the window style: BS_GROUPBOX

 

The button sends WM_COMMAND messages

WM_COMMAND message format:

LOWORD (wParam) //Child Window ID
HIWORD (wParam) //Notification code
lParam //Child window handle

Useful notification codes:

BN_CLICKED

 

Messages for the Check Boxes and Radio Buttons:

This will return non-zero if the button is checked; 0 if NOT checked
iCheck = (int)SendMessage (hwndButton, BM_GETCHECK, 0, 0);

To check the button:
SendMessage (hwndButton, BM_SETCHECK, 1, 0);

To uncheck the button:
SendMessage (hwnd, BM_SETCHECK, 0, 0);

 

Messages for setting the push button:

To cause the button to become depressed:
SendMessage (hwndButton, BM_SETSTATE, 1, 0);

To cause the button to become UN-depressed:
SendMessage (hwndButton, BM_SETSTATE, 0, 0);

 

Functions for all buttons:

SetWindowText (hwnd, pszString);

iLength = GetWindowTextLength (hwnd);
GetWindowText (hwnd, pszBuffer, iMaxLength);

ShowWindow (hwndChild, SW_SHOWNORMAL);
ShowWindow (hwndChild, SW_HIDE);

EnableWindow (hwndChild, TRUE);
EnableWindow (hwndChild, FALSE);

 

Other Notes:

The BM_SETSTYLE message lets you change the button style after the button is created
