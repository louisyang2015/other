Scroll Bars

SetScrollInfo (hwnd, iBar, &si, bRedraw) ;
GetScrollInfo (hwnd, iBar, &si) ;

The iBar is either SB_VERT or SB_HORZ

typedef struct tagSCROLLINFO
{
     UINT cbSize ;     // set to sizeof (SCROLLINFO)
     UINT fMask ;      // values to set or get
     int  nMin ;       // minimum range value
     int  nMax ;       // maximum range value
     UINT nPage ;      // page size
     int  nPos ;       // current position
     int  nTrackPos ;  // current tracking position
}
SCROLLINFO, * PSCROLLINFO ;

Usage:

SCROLLINFO si;
si.cbSize = sizeof(si);

note that you have to set cbSize

before passing "si" to "SetScrollInfo" or "GetScrollInfo", you need to set the "fMask" flag:
SIF_RANGE --- for nMin, nMax
SIF_POS --- for nPos
SIF_PAGE --- nPage
SIF_TRACKPOS --- nTrackPos --- use this only with GetScrollInfo while processing a WM_VSCROLL or WM_HSCROLL message with a notification code of SB_THUMBTRACK or SB_THUMBPOSITION

SIF_ALL --- get/set all values

Windows will limit the maximum scroll bar position to si.nMax - si.nPage + 1

To move the current window very quickly, use:

ScrollWindow(hwnd, delta_x, delta_y, NULL, NULL);
