Polygon

Polygon (hdc, apt, iCount);

SetPolyFillMode (hdc, iMode);

understanding the fill mode: imagine a line drawn from a point in an enclosed area to infinity. The enclosed area is filled if the imaginary line cross an odd number of boundary lines. This default mode is ALTERNATE.

The Brush

HBRUSH hBrush;
hBrush = GetStockObject (GRAY_BRUSH);
SelectObject (hdc, hBrush);

 

hBrush = CreateSolidBrush (crColor);

 


hBrush = CreateHatchBrush (iHatchStyle, crColor);

 

DeleteObject(hBrush);

As usual, do not delete pen or brush while it's selected into the device context.

To get information about a brush:
GetObject (hBrush, sizeof (LOGBRUSH), (LPVOID) &logbrush);
