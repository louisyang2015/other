Numerical Methods for Heat Conduction

One Dimensional Steady Heat Conduction

Say we have a wall of thickness (L) and we want to divide it into pieces that are (Δx = L/5) units thick. The way the division is done is to have 4 elements in the middle and half of an element at each end:

Each of the four pieces in the middle would be Δx units long and the elements at the end would be Δx / 2 units long. Each box is called an "element"

Then we assign a temperature value to each of the six elements.

The temperature values are given physical locations called "nodes". For the four elements in the middle, the node is located in the middle of the element. For the elements on the two ends, the node is located on surface of the object.

For steady state problems, the temperature does not change with respect to time, so each element is not accumulating energy. The energy balance equation for each of the non-boundary nodes:

(heat entering the node) + (heat generated by the node) = 0

Transient Heat Conduction

In transient problems, we have a different set of temperature node values for each point in time. Say { T0, T1 ... Tn } for t = 0, another set of { T0, T1 ... Tn } for t = 1 * Δt, yet another set of { T0, T1 ... Tn } for t = 2 * Δt, and so on.

We figure out the set { T0, T1 ... Tn } at t = 0 first. Then we use that information to figure out the set { T0, T1 ... Tn } at t = 1 * Δt. To go further we use { T0, T1 ... Tn } at t = 1 * Δt to figure out the set at t = 2 * Δt, and so on.

The energy balance equation would look like: (heat entering the node) + (heat generated by the node) = net gain in heat

The left side of the energy balance equation is the same as before. The right side's "net gain in heat" is computed using:

C * ρ * V *
Tnodenew - Tnodeold
Δt

C = specific heat
ρ = density
V = volume
Δt = change in time between Tnodenew and Tnodeold

The T values on the left can be the Tnew or Told . If we use Told , then this is called the "explicit" finite difference formulation. If we use Tnew, then this is called the "implicit" formulation.

When we use implicit formulation, we get one Told term and many Tnew terms for each node, resulting in a system like:

T0old = A0 T0 + A1 T1 + ... + An Tn + Constant0
T1old = B0 T0 + B1 T1 + ... + Bn Tn + Constant1
...

The "old" nodes are known while the new nodes are unknown, so advancing a single time increment means solving a system of equations.

Explicit Formulation and the Stability Criterion

If we use the explicit formulation, there is one Tnew term and many Told terms for each temperature node equation, resulting in a system like:

T0new = A0 T0 + A1 T1 + ... + An Tn + Constant0
T1new = B0 T0 + B1 T1 + ... + Bn Tn + Constant1
...

The old nodes are known --- so it's very straight forward to compute for the new nodes. There is a lot less computation needed to advance a single time increment because we do not have to solve a system of equations.

But for the explicit formulation to work, the coefficients of T0 through Tn must all be greater than or equal to zero. These coefficients will depend on Δt --- so not all Δt values will work. Usually, there is some limit on the largest Δt value that we can use.
