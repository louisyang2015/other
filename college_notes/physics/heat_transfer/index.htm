TI 89 Scripts

After weeks of complacency, I finally bought a TI Connectivity Kit for $14.95 on Saturday, May 10, 2003.

To create a program, use the TI-Graph Link software. To transfer the program, use the Group Explorer software that is part of the TI-Connect software. The TI-Graph Link is really an outdated program that is not compatible with the newer USB connection that I am usign, while the TI-Connect software package does not have software to edit programs.  Note that you can sort of open programs with a normal text editor, but there will be some "garbage" btyes. You can receive file from TI via device explorer, but it does not work well.
