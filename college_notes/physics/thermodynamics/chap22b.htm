Change in entropy during a process depends on just the end points, not on the path between the states. So ∆S = 0 for any cycle like the Carnot cycle or the Otto cycle.

 

In a reversible process, we can go both ways in a pressure-volume graph using the same path, and every point on the path is an equilibrium state.

That is, we can go from state A to state B, or B to A using the same path, and that each point on the path is an equilibrium state.

In real life, there is no such thing as a reversible process because heat can only naturally flow from hot to cold.

For a reversible process, the total entropy of an isolated system remains constant. For a irreversible process, the total entropy of an isolated system always increases.

If two systems interact through an irreversible process, the gain of entropy in one system must be greater than the loss of entropy in the other system.

For example, if a hot reservoir at temperature Th touches a cold reservoir at temperature Tc , and an energy Q flows from the hot reservoir to the cold reservoir, the net change in temperature is:

Adiabatic Free Expansion

Adiabatic process --- this is when no energy enters or leaves the system by heat, so Q = 0 and ∆Eint = -W.

Adiabatic free expansion --- Q = W = ∆Eint = 0. This is basically gas allowed to expand without doing work. The physical model for adiabatic free expansion is to first have a container that is divided into two compartments by a membrane. One compartment is gas while the other compartment is vacuum. The membrane then "leaks" and gas fills both compartment.

The adiabatic free expansion is irreversible because we can not point out a specific path for it. We can measure a starting state of (P1, V1) and an ending state of (P2, V2), but we can not be sure of the intermediate states.

Entropy on a Microscopic Scale

Entropy is really about probability and statistics. For a volume of air there are many ways to place the molecules. So for a single macrostate described by a certain volume, a certain pressure, and a certain temperature, there are a large number of microstates, each differing in how molecules are placed.

Using adiabatic free expansion as an example, let say that each gas molecule takes up volume Vm . There are N gas molecules that starts out taking up volume V1 and later expands to a larger volume V2 .

At volume V1 , the number of ways to place a single molecule is V1 / Vm . In gas, molecules are far apart from one another, so we assume that there are a lot more locations than there are molecules. There are W1 ways to place N molecules, where W1 = (V1 / Vm)N .

Like wise, at volume V2 we have W2 ways to place N molecules, where W2 = (V2 / Vm)N .
