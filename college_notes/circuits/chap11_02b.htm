Bode Plots --- Phase Characteristics

The Bode Phase Graph is a Sum of Terms

Say we are given the transfer function:

Gv(jw) =
10*(1 + 0.1jw)
(1 + jw) (1 + 0.02jw)

The phase of Gv(jw) is:

phase of (1 + 0.1jw) - phase of (1 + jw) - phase of (1 + 0.02jw)

If we know how the graph for each term will look like, then we will know how the graph for the whole thing will look like.

 

 

The Graph for a Single Term

The function:

y = phase of (1 + p * wj),
where p is some positive number

has the following approximation:

w approximate y exact y
(1 / p) / 100 0o 0.573o
(1 / p) / 10 0o 5.7106o
1 / p 45o 45o
10 / p 90o 84.289o
100 / p 90o 59.427o

 

The function:

y = phase of (1 + B*(jw) + (p * jw)2),
where B is some number
where p is some positive number

has the following approximation:

w approximate y exact y
(1 / p) / 100 0o tan-1 { [B / (10p)] / [9999/10000]}
(1 / p) / 10 0o tan-1 { [B / (10p)] / [99/100] }
1 / p 90o 90o
10 / p 180o tan-1 { (10B / p) / -99 }
100 / p 180o tan-1 { (100B / p) / -9999 }

 

Summary:

The term:

(1 + p * wj)

if in the numerator, it means the graph bends upward for 90 degrees over the course of two decades, starting at one decade before w = 1/p. If in the denominator, it means the graph bends downward for 90 degrees over the course of two decades, starting at one decade before w = 1/p.

Changing signs in front of the " 1 " or the positive number " p " can effect whether we bend upward or downward for 90 degrees. If necessary, figure out the details as needed.

 

The term:

(1 + B*(jw) + (p * jw)2)

if in the numerator, it means the graph bends upward for 180 degrees over the course of two decades, starting at one decade before w = 1/p. If in the denominator, it means the graph bends downward for 180 degrees over the course of two decades, starting at one decade before w = 1/p.

Changing signs in front of the " 1 " or the " B " can effect whether we bend upward or downward for 90 degrees. If necessary, figure out the details as needed.

 

The above information tells you when to bend upward or downward. To get starting point for the graph, look for lone " j " terms. Examples:

5 * (a + b*j)
c + d*j

starts at zero degree.

 

5j * (a + b*j)
c + d*j

starts at 90 degrees.

 

5 * (a + b*j)
j * (c + d*j)

starts at -90 degrees.
