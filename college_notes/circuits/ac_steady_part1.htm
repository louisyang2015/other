A Review about Sinusoids

	x(wt) = XM sin(wt)

period = 2Π / w

w = 2Π * f, f is frequency

In EE, it's common to use radians for wt and at the same time use degrees for the phase shift.

	x(t) = XM sin(4t + 30o)

The "4t" is in radians, while the "30" is in degrees.

Basic Analysis Approach using Differential Equations

Example problem: Find i(t) if v(t) = VM cos wt

L
d i(t)
dt
+ i(t) * R = v(t) = VM cos wt

v(t) is a cosine function --- so the current and voltage everywhere in these circuits would be a cosine function too. The reason is that the inductor and capacitor equations are: v = L (di/dt), and i = C (dv/dt). So if v is sine, then i is cosine. If v is cosine, then i is sine. A sine function is just a cosine function with a phase shift of 90 degrees.

Knowing that i(t) will be in the form: 

	i(t) = A cos(wt + Φ)

we can try to substitute this "A cos(wt + Φ)" into the differential equation and get the constants "A" and "Φ". Substituting "A cos(wt + Φ)" into the differential equation will result in:

	A*w*L * -sin(wt + Φ) + R*A * cos(wt + Φ) = VM * cos(wt)

It's possible to rearrange things to match up the left side with the right side and figure out "A" and "Φ" --- but it is very tedious. We start by:

	sin(wt + Φ) = (sin wt)(cos Φ) + (cos wt)(sin Φ)
	cos(wt + Φ) = (cos wt)(cos Φ) - (sin wt)(sin Φ)

and the left side eventually becomes:

	something * sin(wt) + something * cos(wt) = VM * cos(wt)

A much easier approach that substituting in A cos(wt + Φ):

Recall that:

	ejwt = cos wt + j * sin wt

Instead of solving

L
d i(t)
dt
+ i(t) * R = VM cos wt

Solve:

L
d i(t)
dt
+ i(t) * R = VM ejwt

and the the real part of the answer.

The substitution to use is:

	i(t) = A * ej(wt + Φ)

The reason this substitution works better than "A cos(wt + Φ)" is that the derivative of "ej(wt + Φ)" will be a constant times "ej(wt + Φ)" --- and this makes term matching easy. In contrast, the derivative of "A cos(wt + Φ)" is a sine term. Also, factoring out the "Φ" from the (wt + Φ) is more easily done with the esomething format.
