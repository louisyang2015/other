Transfer Function in Circuit Analysis

Superposition

If the output is generated by more than one input source in a network, we can have a transfer function for each source, and add them together to get the transfer function for the network.

Because input is restricted to be zero before time zero, the capacitors and inductors at time zero do not have a initial voltage or current associated with them. When the book ask us to get the transfer function for a circuit, we can just transform a capacitor to 1/(sC) and an inductor to sL without having to worry about the initial current/voltage of capacitors and inductors.

 

Poles

The denominator portion of the transfer function is from the left side of the differential equation that describes the system. The numerator portion is from the right side of the differential equation. The denominator is about the system's natural response, while the numerator is about the forcing function for the system.

The denominator portion of the transfer function is the characteristic equation of the differential equation for the system. This characteristic equation can result in a solution that falls into the three categories: over damped, under damped, and critically damped. Chapter 6 went over the details.

The roots of the denominator portion are called poles. From the perspective of the transfer function, if the denominator has a quadratic function that has unequal and real roots (poles), then the system is over damped. Having a repeated root (pole) means the system is critically damped. Having a pair of conjugate root (pole) means the system is under damped.

So the poles of a transfer function can tell you if the natural response to the system is over damped, critically damped, or under damped.

If there is a positive pole, it means the denominator has a term like (s - 2), and the natural response would then have a term like e2t , which goes off to infinity as time increases. Having a positive pole therefore means the system is unstable and normally would not happen in electrical circuits.

Finally, poles eventually show up as the time constants in the natural response solution. So smaller poles will mean that the circuit is more responsive and approach steady state faster.

Obtaining Steady Response from the Transfer Function

Getting the steady state just means ignoring all those e-2t terms that will disappear over time.

If the input is sinusoidal, there is a fast way to get the steady state response from a transfer function. We evaluate the transfer function H(jw) at the proper w. This will produce a complex number with magnitude M and phase P.

If the input is:

A * cos(wt + 45o)

the steady state output will be:

A * M cos(wt + 45o + P)

Example:

H(s) =
s2
3s2 + 4s + 4

If the forcing function (input) is:

10 * cos(2t) * u(t)

H(j2) = 0.354 /45o

The steady state response is:

voss(t) = 3.54 cos(2t + 45o)
