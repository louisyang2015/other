BIOS Interrupt 10h, Service 6h

This service is used to scroll the screen upward

For a normal screen, the upper left corner is (0, 0) and the lower right corner is (79, 24). Another possible height is to have 50 lines per screen.

mov CL, upper_left_x
mov CH, upper_left_y

mov DL, lower_right_x
mov DH, lower_right_y

mov AL, num_lines_to_scroll_upward_by    ;  if 0, screen is cleared

mov AH, 06h
int 10h            ; execute interrupt 10h, service 6h
