Make File:

The make file (which by default is named "makefile") specifies dependency information among files, and then gcc use that dependency information to direct the linker.

eatlinux : eatlinux.o

this line is saying that to generate the executable file "eatlinux", we first need to have the file for eatlinux.o.

If one file depends on multiple files:

linkbase : linkbase.o linkparse.o linkfile.o

Following the dependency line is a line telling us how to generate the first file using the dependency files:

eatlinux : eatlinux.o
    gcc eatlinux.o -o eatlinux
