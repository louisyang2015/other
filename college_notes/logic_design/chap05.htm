Minterm and Maxterm Expansions

An example on page 91. Find the minterm expansion of:

    f(a, b, c, d) = a'(b' + d) + acd'

Minterm expression:

f = ∑m(0, 1, 2, 3, 5, 7, 10, 14)

f = a'b'c'd' + a'b'c'd + a'b'cd' + a'b'cd + a'bc'd + a'bcd + ab'cd' + abcd'

Maxterm expression:

f = ∏M(4, 6, 8, 9, 11, 12, 13, 15)

f = (a + b' + c + d) (a + b' + c' + d) (a' + b + c + d) (a' + b + c + d')
(a' + b + c' + d') (a' + b' + c + d) (a' + b' + c + d') (a' + b' + c' + d')

note that for maxterms the encoding for the literals is opposite of the encoding used for the minterms: 0100 = a + b' + c + d
