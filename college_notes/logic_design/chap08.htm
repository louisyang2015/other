Multi-Level Gate Networks

Multi-Level Gate Networks

The maximum number of gates cascaded in series between a network input and the output is referred to as the number of levels of gates. Inverters don’t count.

Other Types of Logic Gates

majority gate – has an odd number of inputs, and its output is 1 iff a majority of its inputs are 1;

minority gate – has an odd number of inputs, and its output is 1 iff a minority of its inputs are 1

Functionally Complete Sets of Logic Gates

The AND-OR-NOT is a functionally complete set of gates. The NAND gate is also functionally complete.

X NAND X = (X * X)’ = X’ + X’ = X’ = NOT X
(A NAND B) NAND (A NAND B) = ((A * B)’ * (A * B)’)’

= (A * B) + (A * B) = A * B
(A NAND A) NAND (B NAND B) = A’ NAND B’ = (A’ * B’)’ = A * B

To notice how a gate is functionally complete, study the truth table for the gate.

 

Eight Basic Forms

There are a total of eight basic forms for two level networks. You can cycle through four of those forms by complementing the minterm Boolean equation twice.

AND - OR F = A + BC' + B'CD
NAND - NAND F = [(A + BC' + B'CD)']'
F = [A' (BC')' (B'CD)']'
OR - NAND F = [A' (B' + C) (B + C' + D')]'
NOR - OR F = A + (B' + C)' + (B + C' + D')'

The four remaining forms can be cycled through by complementing the maxterm version of the Boolean equation twice.

Mixed Logic and Polarity Indication

Inside of a logic device we speak of zeros and ones. But outside of a logic device we speak of high and low voltages.

active high - positive logic; this means that the active or true state corresponds to a high voltage

active low - the active or true state corresponds to a low voltage

Consider a variable "ENABLE" which is used to enable the operation of a logic circuit. If ENABLE is active high, this implies that a high voltage will enable the logic circuit, and if ENABLE is active low, a low voltage will enable the circuit.

A few naming conventions:

Active High Active Low
ENABLE(H) ENABLE(L)
ENABLE.H ENABLE.L
ENABLE+ ENABLE-
ENABLE ~ENABLE
ENABLE /ENABLE

On the schematics, a half arrow symbol is used to denote active low input or output. This half arrow is similar to the notation circle --- but the circle is a logical description while the half arrow is an electrical description.
