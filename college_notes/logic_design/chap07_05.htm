Map Entered Variable Variation of the K-Map

This is just a scheme to fit more variables onto a K-Map

Example:

F(A, B, C, D) = A'B'C + A'BC + A'BC'D + ABCD + (AB'C)

where the last term AB'C is a don't care.

There are four variables, but we can fit it onto a three variable K-Map by using:

    A
    0 1
BC 00
   
1 x
1 D
D  
01
11
10

The "D" acts as a Boolean flag --- the D minterms are 1 if D is 1, the D minterms are 0 if D is 0. There are two sets of K-Maps, and we satisfy the function f by satisfying both K-Maps.

The D = 0 K-map:

    A
    0 1
BC 00
   
1 x
1  
   
01
11
10

is satisfied by " A'C "

The D = 1 K-map:

    A
    0 1
BC 00
   
1 x
1 1
1  
01
11
10

is satisfied by: " C + A'B "

The final answer:

A'C + D(C + A'B)
= A'C + CD + A'BD
