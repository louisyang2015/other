Algebraic Simplification

The Consensus Theorem

XY + X'Z + YZ = XY + X'Z

The YZ is called the consensus term. It comes from multiplying the XY term with the X'Z term while leaving out the X and X'.

To prove the consensus theorem (or avoid remembering it), use: XY + X'Z + YZ(X + X')

The dual of the consensus theorem:

(X + Y)(X' + Z)(Y + Z) = (X + Y)(X' + Z)

This time, the Y + Z is the consensus term. It is arrived at by adding the X + Y and X' + Z term while leaving out the X and X'.

 

Proving Validity of an Equation

It is permissible to perform the same operation on both sides of the equation provided that operation is reversible." The key word is "reversible --- many operations that are reversible in arithmetic algebra are not reversible in Boolean algebra.

The cancellation law is not true for Boolean algebra. Attaching AND, OR terms to both sides of a Boolean equation is not used in proofs. For example:

1 + 0 = 1 + 1, but 0 ≠ 1
1 * 0 = 0 * 0, but 0 ≠ 1

However, the converse of the above is true:

If y = z, then x + y = x + z
If y = z, then xy = xz

So if you know for sure two expressions are equal, you can add the same stuff to both sides to maintain that equality.

If you do not know for sure two expressions are equal, you cannot add the same stuff to both sides to proof equality.
