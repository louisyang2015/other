BJT Fundamentals

Overview

For a BJT, the outer two regions are not interchangeable. The emitter has a different geometry and is typically more heavily doped than the collector.

Basic current and voltage laws:

IE = IB + IC

VEB + VBC + VCE = 0

There are three terminals. If two terminals serve as input, and two terminals serve as output, there must be a terminal common to both input and output. The common emitter is the most widely employed configuration, with the common base finding occasional utilization. The common collector is seldom if ever used and will be henceforth neglected.

Active Mode --- This is the mode used in amplifiers.

Saturation Mode --- This mode results in high current, low voltage

Cutoff Mode --- This mode results in low current, high voltage

Inverted Mode --- This mode is the active mode with the roles of the emitter and collector swapped.

Active Bias Current Flow

The main current activity is IEp and ICp . Holes move from the p+ emitter to the base because there is a forward bias there. The key to the BJT is that the base is very narrow, so most of the holes will diffuse to the B-C junction, where there is a reverse bias. Holes are minority carriers in the n doped base, so the holes get swept across the B-C junction.

A small number of holes will recombine in the base, and this is creates the current IB2 .

Two other currents are majority carriers going from the base to the emitter due to the forward bias at E-B junction, and minority carriers going from the collector to the base due to the reverse bias at the C-B junction.

The reverse bias R-G current is ignored for now.

The combined picture:

IE = IEp + IEn

IC = ICp + ICn

IB = IB1 + IB2 + IB3

IEn << IEp

ICn << ICp

Common Base d.c. Current Gain ( αdc )

In the common base configuration

IC = αdc IE + ICB0

where αdc is the common base d.c. current gain and ICB0 is the collector current that flows when IE = 0 . It turns out that:

αdc = y αT
